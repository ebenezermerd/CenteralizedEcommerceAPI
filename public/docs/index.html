Introduction

Welcome to the Koricha E-commerce API. This API provides endpoints for managing products, orders, customers, and other e-commerce related functionality.

Base URL: http://api.koricha-ecommerce.com

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

APIs for managing authentication

Register a new user

Create a new user account with the provided details.

Example request:
curl --request POST \
    "http://api.koricha-ecommerce.com/api/auth/sign-up" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \

    \"role\": \"customer\",
    \"firstName\": \"John\",
    \"lastName\": \"Doe\",
    \"email\": \"john@example.com\",
    \"phone\": \"+1234567890\",
    \"sex\": \"male\",
    \"address\": \"123 Main St\",
    \"password\": \"Secret123!\",
    \"confirmPassword\": \"eius\",
    \"verified\": false,
    \"recaptchaToken\": \"sunt\",
    \"agreement\": true,
    \"companyName\": \"Tech Corp\",

    function run () {
